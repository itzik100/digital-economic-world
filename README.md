# עולם דיגיטלי כלכלי קהילתי

תיקיית פרויקט ראשית למשחק עולם פתוח קהילתי עם כלכלה וירטואלית, משאבים טבעיים, רובוטיקה, בנייה, שוק וטוקנים פנימיים.

## משפט כיוון
אנחנו לא בונים משחק טוקנים. אנחנו בונים עולם דיגיטלי חי שמתחיל מהטבע, עובר למשאבים, רובוטיקה, בנייה, מסחר, נכסים וכלכלה.

## מבנה התיקייה

- 00_חומרי_מקור — מסמכים ורעיונות ישנים
- 01_חזון_וחוקי_עולם — חוקי בראשית, טבע, משאבים ותהליכים
- 02_MVP_ויזואלי — מה צריך לראות בגרסה הראשונה
- 03_משאבים_וכלכלה — עץ, אבן, ברזל, מים, אדמה, מחירים ומסחר
- 04_רובוטיקה_ובנייה — רובוטים, מדפסת בטון, בניית בתים ותעשייה
- 05_אווטארים_רכבים_ומפה — דמות, רכבים, אזור ראשון ומפה
- 06_טוקנים_ושוק — טוקנים פנימיים, חנות, שוק ומסחר עתידי
- 07_טכנולוגיה_Unity_Backend — Unity, Backend, Database, Realtime
- 08_אדמין_וקהילה — ניהול משתמשים, משימות, טוקנים וקהילה
- 09_משימות_פיתוח — רשימת משימות לביצוע
- 10_מסמכי_שיווק_ומשקיעים — הצגה עתידית לקהילה/שותפים/משקיעים
