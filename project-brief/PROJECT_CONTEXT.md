# Project Context — עולם דיגיטלי כלכלי קהילתי

## מה הפרויקט?

עולם דיגיטלי פתוח שמסמלל כלכלה אמיתית בתוך סביבת משחק. השחקנים חיים, עובדים ומסחרים בתוך עולם וירטואלי עם כלכלה חיה.

## עקרון הליבה

אנחנו לא בונים משחק טוקנים. אנחנו בונים עולם דיגיטלי חי שמתחיל מהטבע, עובר למשאבים, רובוטיקה, בנייה, מסחר, נכסים וכלכלה.

## מרכיבי העולם

- **משאבים טבעיים** — עצים, אבנים, מים, אדמה, מחצבים
- **רובוטיקה ובנייה** — רובוטים אוטונומיים, מדפסת בטון, בניית מבנים
- **שוק פנימי** — קנייה ומכירה בין שחקנים, מחירים דינמיים
- **טוקנים וירטואליים** — מטבע פנימי למסחר, שכר ופרסים
- **קהילה** — משתמשים, תפקידים, שיתוף פעולה
- **אווטארים ורכבים** — דמות אישית, תחבורה, ציוד

## טכנולוגיה

- **Frontend**: Vite + Vanilla JS
- **Backend**: Node.js + Express + Prisma
- **DB**: PostgreSQL (Docker)
- **Realtime**: Socket.IO
- **3D/Visualize**: Unreal Engine 5 (שלב מתקדם)

## חזון עתידי

בהמשך — אחרי MVP יציב — ייתכן חיבור לקריפטו אמיתי, NFT לנכסים ייחודיים, ושוק חיצוני. אך זה לא חלק מהשלב הנוכחי.
