## Pipeline לטיפול ב-Assets

```
1. רכישה / הורדה
2. Validate (מידות, פורמט, LODs)
3. Import לפרויקט UE5
4. Material setup + חיבור texture maps
5. LOD בדיקה
6. Collision setup
7. Catalog ב-Content Browser
8. Tag + metadata
```
