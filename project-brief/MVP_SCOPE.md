# MVP Scope

## מה נכנס ל-MVP הראשון

- [ ] רישום והתחברות שחקנים
- [ ] אווטאר בסיסי עם מיקום בעולם
- [ ] איסוף משאבים (עץ, אבן, מים)
- [ ] אינוונטורי אישי
- [ ] שוק פנימי — קנייה ומכירה בין שחקנים
- [ ] טוקן וירטואלי פנימי (לא קריפטו)
- [ ] רובוט בסיסי שאוסף משאבים
- [ ] מפה של אזור ראשון
- [ ] ממשק ניהול (Admin)
- [ ] משימות בסיסיות (Quests)

## מה לא נכנס ל-MVP

- קריפטו אמיתי
- NFT
- עולם פתוח ענק
- כסף אמיתי / תשלומים
- מובייל
- Unreal Engine 5 בפרודקשן
- מולטיפלייר בקנה מידה גדול
- מערכת כלכלה מורכבת עם אינפלציה/דפלציה
