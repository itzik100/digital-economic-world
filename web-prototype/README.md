## מטרת הדמו
- בדיקת פיצ׳רים ורעיונות מהר
- דשבורד / ממשק ניהול עתידי
- Pixel Streaming placeholder עד שה-UE5 מוכן
- בדיקת Backend API

## מה כלול כאן
- Three.js world (3D prototype)
- Market system (live price ticker)
- Globe map (3D interactive)
- Building system (basic)
- HUD + panels
- Google OAuth (dev mode)

## מה לא כלול (יהיה ב-UE5)
- Terrain ריאליסטי
- רכב אמיתי
- Physics
- Assets איכותיים
- Third Person camera מלאה
